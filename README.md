# MyFlightController

<img width="976" height="821" alt="Screenshot 2026-07-05 180633 (1)" src="https://github.com/user-attachments/assets/d264c197-2e49-44dd-beed-56dd9d1c45c9" />

So I made a custom flight controller which can be used for an RC Plane . it is based on STM32 and The Flight Controller  has 2 IMU's and a barometer for precise data collection. It has connectors for 2 Servos to control the wind / 2 ESCs and a reciever . It will work on a 12V battery. 

## Features
- Base on  STM32F401CBUx 
- Has dual IMU (MPU_6050)(ICM-20948)
- Has a Barometer (BMP280)
- Works on 12 V battery
- Has multiple voltage regulators for  (5v , 3.3v , 1.8v )
